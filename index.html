Topology 3D Help

  • Create vertex: Simply by using the drag and drop icons of the nodes, or pressing Shift +left-click
  • Create edge: Select the first vertex by clicking on it. Shift +left-click on another vertex (different than the selected one).
  • Delete a vertex/edge: dwhen vertex/edge is selected.
  • Freeze/unfrees a vertex: press f when the node is selected (a frozen node does not move in live mode)
  • Undo vertex deletion: Click on the Undo button. Only the last deleted vertex can be recovered.
  • Switch to automatic node positioning: Menu Layout -> Live
